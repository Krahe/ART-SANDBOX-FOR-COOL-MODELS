Key Typography Principles:

1. HIERARCHY AND WEIGHT:
   - Thicker characters = more visual weight
   - Use weight to establish importance
   - Block fills (█) are heaviest
   - Outlines (╔═╗) are lighter
   - Mixed for emphasis

2. SPACING AND BREATHING:
   - Leave space around important text
   - Use padding (░▒▓) for depth
   - Wider letter spacing = more formal
   - Tighter spacing = more compact/tech

3. CONSISTENCY:
   - Choose one style per piece
   - Mix styles only for contrast (title vs. body)
   - Keep line weights consistent within a style
   - Align elements for professional look

4. DECORATIVE BALANCE:
   - Don't over-decorate
   - Ornaments should enhance, not distract
   - Symmetry creates stability
   - Asymmetry creates energy

5. PRACTICAL SIZING:
   - Small: Single-line characters for body text
   - Medium: 3-5 line height for sub-headers
   - Large: 7+ line height for main titles
   - Consider terminal width (132 columns max)

6. CHARACTER SELECTION:
   - ║═╔╗╚╝ for clean, technical
   - ░▒▓█ for depth and dimension
   - ◆◊★✦ for decorative accents
   - ┌─┐└│ for simple frames

TYPOGRAPHY WORKFLOW:

1. Choose style (block, outline, decorative)
2. Determine size (header, sub-header, body)
3. Add framing/background if needed
4. Consider depth (shadow, 3D, gradient)
5. Test legibility at target width
6. Refine spacing and alignment
